# CLI To-Do Application

A simple Command Line To-Do application built with Python that allows users to manage daily tasks directly from the terminal.

## What this program does

This application lets users create and manage a task list.  
Tasks are saved in a file so they remain available even after the program closes.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Saves tasks in a file

## Technologies Used

Python

Modules:
- json
- os

## How to Use

1. Run the program.
2. Choose an option from the menu.
3. Add, delete, or mark tasks as completed.
4. Tasks will be saved automatically.

## Example

Menu:

1. Add Task  
2. View Tasks  
3. Mark Task as Done  
4. Delete Task  
5. Exit
